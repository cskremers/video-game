Expedition Protocol

Design planner · review on phone · export back to Cursor

Core Concept

2D co-op expedition game: clear procedural sites (B), solve coordination puzzles (D), upgrade a shared outpost under rising threat (C). Steam · 2–8 players · flexible session length.

B — Clear dungeon C — Build & defend D — Solve together

Session Loop

Outpost Hub → Prep → Deploy → Explore Rooms → Combat / Puzzle / Rest → Loot → Extract? ↓ yes ↓ push deeper Hub upgrades ← Threat meter ← More rooms → Boss

Session Modes

ModeTimeStructure
Sortie10–20 min3–5 rooms, 1 objective, guaranteed extract
Expedition30–45 minFull floor, boss, meaningful loot
Campaign night60–90+ minMulti-floor + hub pressure event mid-run

Co-op Roles (light)

  • Vanguard — front line, block/taunt
  • Striker — burst damage
  • Support — heal, buff, revive
  • Tech — deployables, hacks, puzzle terminals

2 · Combat System Spec

Three combat directions. All support 2–8 co-op and host-authoritative netcode.

Option A — Dodge-Forward (Hades-like)

fastskill expression

Player verbs

  • Move + aim attack (8-dir or mouse)
  • Dash — i-frames 0.2s, stamina cost
  • Primary / secondary weapon, 1 ability, 1 ultimate (cooldown)

Damage model

StatBase
Player HP100 (+ gear)
Light hit8–12
Heavy telegraph25–35 (1.2s windup)
Revive window8s downed → dead → spectate

Enemy telegraph language

  • Red floor AoE = dodge
  • Enemy flash white = parry window (Vanguard only)
  • Projectile = small tint + travel line

Example enemies (Industrial)

Scrap Drone (swarm), Arc Sentry (ranged), Crusher (slow heavy), Furnace Golem (elite)

Option B — Tactics & Positioning (Alien Swarm-like)

methodicalsquad play

Player verbs

  • Move, aim, hold position (accuracy bonus)
  • Class gadgets: turret, med beacon, shield wall, scanner
  • Reload + ammo types (AP vs shielded)

Damage model

StatBase
Player HP120 (+ armor slots)
Chip damage5–8 sustained
Focus fire bonus+15% same target
Revive12s channel by Support

Telegraph language

  • Directional cone overlay = reposition
  • Elite aura color = required damage type

Example enemies

Grunt horde, Shieldbearer, Spitter (zone deny), Brood anchor (objective kill)

balanced

3 · Hub Upgrade Tree

Three outpost fantasies. All use Threat meter (rises each expedition; triggers events at 25/50/75/100).

Option A — Minimal Outpost (C-lite)

low scope v1

Single screen hub, 4 module slots, no active defense minigame.

ModuleCostEffect
Workshop200 scrap+1 weapon mod slot
Med Bay150 scrap+20% revive speed
Sensor Array250 scrapReveal 1 room type on map
Bulkhead300 scrapThreat rise −10% per run

Threat 100: next deploy starts with "Emergency Extract" modifier (harder, bonus loot).

Option B — Living Base (C-medium)

more persistence

6 modules, soft degradation, repair tasks between runs.

TierModuleUnlock
1Power CoreEnables all other modules
1Supply Cache+starting consumable
2Fab Benchcraft mods from expedition drops

Threat 75+: "Incursion" — deploy with hub debuff until cleared room in expedition.
